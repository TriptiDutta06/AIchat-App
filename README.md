# 🤖 AI Chat Application (Gemini 2.0)

A sleek, responsive AI Chat interface built with vanilla web technologies. This application connects directly to the **Google Gemini API** to provide intelligent, real-time responses.

---

## 🚀 Features
* **Real-time Interaction:** Instant responses from the Gemini 2.0 Flash model.
* **Modern UI:** A clean, dark-themed aesthetic inspired by Google Gemini.
* **Responsive Design:** Works perfectly on both Desktop and Mobile browsers.
* **Keyboard Support:** Press `Enter` to send messages instantly.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **API:** Google Generative AI (Gemini API)
* **Version Control:** Git & GitHub
